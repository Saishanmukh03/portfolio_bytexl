Projects

Brain Tumor Detection

This project assists medical professionals in detecting and classifying brain tumors using MRI images by employing a modified VGG19 architecture. The deep learning model processes scans to identify and accurately classify different types of tumors, aiding in timely and precise diagnoses. By recognizing subtle patterns in MRI scans, it distinguishes between benign and malignant tumors with high precision.

Railway Ticket Booking System

Developed a Railway Ticket Booking System using HTML and PHP. This web application allows users to book railway tickets online, featuring user registration, train schedule viewing, seat selection, booking confirmation, and payment integration. An admin interface is included for managing train schedules, availability, and user information, streamlining the booking process and enhancing user experience.
